Tutorial
This tutorial will guide you on how to setup Moniajo, the various functions it can perform and how to use it.

So, let's get started!!!
1. Setup

Step 1: Before powering on the bot for the first time, locate and remove the memory card from the bot.

Step 2: Insert the memory card in the card reader (provided in the pack) and plug it into a Laptop/Computer.

Fig. 1.2.1

Fig. 1.2.2

Fig. 1.2.3
❮ ❯

Step 3: Open Windows Explorer and locate the drive named 'boot' and open it.

Step 4: Create a new file with the name wpa_supplicant.conf in the boot folder

Step 5: Open wpa_supplicant.conf and insert the following content into the file. 

            
    ctrl_interface=DIR=/var/run/wpa_supplicant  
    update_config=1
    country=<Insert country code here>
              
    network={
      ssid="<Name of your WiFi>"
      psk="<Password for your WiFi>"
    }

For example, if your country is India, the name of your Wi-Fi is User and the password is password, then this is what you file would look like. 
            
    ctrl_interface=DIR=/var/run/wpa_supplicant  
    update_config=1
    country=IN
              
    network={
      ssid="User"
      psk="password"
    }

A list of country codes is available here.
Step 6: Eject the drive. Insert it back into the bot.


2. Usage

To use Moniajo, download the app from here.
Moniajo runs a 3.7 volt PowerPack.
To switch on Moniajo, press the button on the left side of the bot once.
To switch off Moniajo, press the switch on the PowerPack twiceh.
Moniajo's features can be accessed as shown below.

Gesture Control


Every screen of the Android Application contains a button on the top right corner of the screen which can be used to access the gesture control feature after the Device has been connected to the bot
Chatbot


The Home screen of the Android Application is for accessing the Chat bot. To use the chat bot follow the steps given below.
Step 1: Click the "Connect" button on the screen to connect to Moniajo
Step 2: Post a query either by using the Text-To-SPeech button or by typing in the text box provided and click send.
Camera Feed


The surveillance module can be accessed by navigating to the "Camera Feed" section using the navigation side bar.
The slider provides Zoom in and Zoom out functionality.
Arm Control


The Mechanical Arm can be accessed by navigating to the "Arm Control" section using the navigation side bar.
The Mechanical Arm can be controlled by using the sliders on the page.

3. Charging

To charge Moniajo, follow the steps given below.

Step 1: Switch off the bot
Step 2: Connect the micro USB cable to the PowerPack.
About Us
I wish I could control everything with my hands, sitting in my chair. Everyone wishes for that, right?
Being a programmer we were tired of sitting and coding for hours without seeing a "Physical output" of what we were doing. We wanted a physical product in our hands, a product which would help people with simple tasks like surveillance or someone you can have to talk to you when you're alone.

And hence, we came up with Moniajo - A gesture controlled robot with a chatbot, robotic arm and a live camera feed.

Features include:
  • Live Camera Feed
  • State of the art Chatbot
  • Motion Control via Mobile Application
  • 4 Axis Robotic Arm
